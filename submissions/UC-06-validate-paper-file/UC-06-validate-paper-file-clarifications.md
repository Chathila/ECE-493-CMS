# Clarification Report: UC-06 Validate Paper File Format and Size

**Date**: 2026-02-05

## Questions & Answers

- Q: What file formats are supported?  
  A: PDF and DOCX.
- Q: What is the maximum allowed file size?  
  A: 20 MB.
- Q: How should corrupted/unreadable files be handled?  
  A: Reject upload and show a file cannot be processed error message.
