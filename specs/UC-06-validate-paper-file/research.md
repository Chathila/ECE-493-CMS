# Phase 0 Research: Validate Paper File Format and Size

## Decision: Supported formats are PDF and DOCX

**Rationale**: Common conference submission formats and aligned with UC-04.

**Alternatives considered**: PDF only; PDF/DOCX/LaTeX.

## Decision: Maximum file size is 20 MB

**Rationale**: Suitable for typical manuscript sizes without excessive storage
costs.

**Alternatives considered**: 10 MB or 50 MB.

## Decision: Corrupted/unreadable files are rejected

**Rationale**: Files that cannot be parsed should not proceed in the workflow.

**Alternatives considered**: Accept and flag for manual review.
