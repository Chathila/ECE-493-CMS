# Clarifications: UC-10 Accept or Reject Review Invitation

**Created**: 2026-02-06

## Questions and Answers

1. **Q**: Can a referee change their response after submission?
   **A**: No, only one response per invitation; first response locks it.
2. **Q**: What should happen when a referee opens an expired invitation?
   **A**: Block access and show an expired message.
3. **Q**: What content should be included in editor notifications?
   **A**: Include only the accept/reject decision.
