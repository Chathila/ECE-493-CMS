# Phase 0 Research: View Conference Registration Prices

## Decision: Pricing visible to guests and registered users

**Rationale**: Clarified requirement specifies visibility to both guests and
registered users.

**Alternatives considered**: Restrict to logged-in users.

## Decision: Unavailable pricing message is minimal

**Rationale**: Clarified requirement states "pricing information is currently
unavailable" without extra guidance.

**Alternatives considered**: Provide check-back guidance.

## Decision: Retry-later message includes retry guidance

**Rationale**: Clarified requirement instructs users to try again later.

**Alternatives considered**: Generic error message.
