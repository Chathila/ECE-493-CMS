# Phase 0 Research: Edit Conference Schedule

## Decision: Conflicts highlighted inline

**Rationale**: Clarified requirement specifies inline conflict highlighting for
conflicting sessions.

**Alternatives considered**: Global error banner or separate conflict list.

## Decision: Validation errors identify missing/invalid fields

**Rationale**: Clarified requirement requires field-level guidance.

**Alternatives considered**: Generic validation errors.

## Decision: Retry messages instruct editor to retry later

**Rationale**: Matches extension guidance without adding support flow.

**Alternatives considered**: Retry now; contact support.
